# \# TextJedi

# 

# Interpreter and lexical analyzer implementation for the TextJedi programming language.

# 

# \## Features

# 

# \- Lexical analyzer

# \- Interpreter

# \- Integer and text operations

# \- Built-in functions

# \- File input/output

# 

# \## Run

# 

# ```bash

# TextJedi myProg

# ```

# 

# Runs `myProg.tj`.
